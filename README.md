# Тестовое задание

HTML/CSS/JS-кодер (верстальщик)

### Запустить продакшн

```
git clone https://github.com/angl1chanin/ibrush-test-layout
cd ibrush-test-layout/dist
open index.html
```

```
git clone https://github.com/angl1chanin/ibrush-test-layout
cd ibrush-test-layout
npm install
npm run start
```
